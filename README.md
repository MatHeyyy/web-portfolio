# Web Portfolio

A personal portfolio website showcasing my projects, skills, and experience as a Computer Science student at the University of Dundee.

## Github Pages

View the portfolio at: https://matheyyy.github.io/web-portfolio/index.html

## Features

- **Responsive Design**: Fully responsive layout that works on desktop, tablet, and mobile devices
- **Modern UI**: Dark theme with gradient backgrounds and smooth animations
- **Three Main Pages**:
  - **Home**: Welcome section with featured projects
  - **About**: Personal introduction, skills, and contact information
  - **Portfolio**: Complete showcase of all projects

##  What I Used

- **HTML5**: Semantic markup
- **CSS3**: Custom styling with CSS variables for theming
- **Bootstrap 5.3.2**: Responsive grid system and components
- **Bootstrap Icons**: Icon library for enhanced UI
- **Google Fonts**: Space Grotesk font family

##  How To Run

### Prerequisites

A modern web browser (Chrome, Firefox, Safari, or Edge)

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/MatHeyyy/web-portfolio.git
   ```

2. Navigate to the project directory:
   ```bash
   cd web-portfolio
   ```

3. Open `index.html` in your web browser or use a local server:
   ```bash
   # Using Python 3
   python -m http.server 8000
   
   # Using Node.js
   npx http-server
   ```
